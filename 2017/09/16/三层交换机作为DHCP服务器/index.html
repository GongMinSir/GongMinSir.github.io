三层交换机作为DHCP服务器

发表于 2017-09-16 | 分类于 路由

TOP图


详细配置

VLAN10区域配置

S2

system
int g0/0/1
port link-type access
port default vlan 10
int g0/0/2
port link-type trunk
port trunk allow-pass vlan all

VLAN20区域配置

S3

system
int g0/0/1
port link-type trunk
port trunk allow-pass vlan all
int g0/0/2
port link-type access
port default vlan 20

三层交换机的配置

S1

system
vlan 10
quit
vlan 20
quit

int vlan 10
ip add 192.168.1.1 255.255.255.0
quit
int vlan 20
ip add 192.168.2.1 255.255.255.0
quit

int g0/0/1
port link-type trunk
port trunk allow-pass vlan all
quit
int g0/0/2
port link-type trunk 
port trunk allow-pass vlan all
quit

dhcp enable
ip pool 10
network 192.168.1.0 mask 255.255.255.0
dns-list 202.196.32.1
gateway-list 192.168.1.1
lease day 10
quit

dhcp enable
ip pool 20
network 192.168.2.0 mask 255.255.255.0
dns-list 202.196.32.1
gateway-list 192.168.2.1
lease day 10
quit

int vlan 10
dhcp select global
quit
int vlan 20
dhcp select global

知识点

在之前的不同见vlan通信的top基础上，把三层交换机作为DHCP服务器，配置在三层交换机上配置，配置完之后把PC换为DHCP就可以了。
